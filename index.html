Building the page

Notes are untangled from source files and rendered as markdown. Diagrams are rendered by mermaid.js. Page is styled with mvp.css.

It looks a bit like this:

flowchart LR; subgraph "source files" src1.py src2.py other_src.py[....py] end untangler([untangler]) src1.py --> untangler src2.py --> untangler other_src.py --> untangler subgraph "external resources" mermaid([mermaid.js]) mvp([mvp.css]) cdn((cdn)) mermaid --> cdn mvp --> cdn end untangler --> index.html cdn --> index.html
